### Reporting Bugs
Before creating a bug report, please **search the existing issues** to see if the problem has already been reported. If you find a match, add a üëç reaction or a comment with additional details.

If you need to open a new issue, please include:
* A clear, descriptive title.
* Steps to reproduce the behavior.
* Expected vs. actual results.
* Your environment (OS, Julia version, etc.).

### Suggesting Enhancements
We love new ideas! To suggest an enhancement:
1. Check if the feature has already been suggested.
2. Open an issue with the tag `enhancement`.
3. Explain **why** this feature would be useful and how it should work.
